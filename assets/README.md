# Assets Directory

## VLC Icon
To display the VLC icon properly, please add a VLC logo image file:
- **File name**: `vlc_icon.png`
- **Location**: Place it in this `assets/` directory
- **Recommended size**: 256x256 pixels or similar
- **Format**: PNG with transparency

You can download the official VLC cone icon from:
- VLC official website: https://www.videolan.org/
- Or use any VLC logo image you prefer

The app will fall back to a generic video library icon if the VLC icon is not found.
